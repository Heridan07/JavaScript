O Estudo dos Eventos DOM

Um evento é tudo aqueli que pode acontecer dentro de uma div. Ex: Passar o mouse em cima e a div mudar de cor.

MDN Web docs é um site com listas de eventos DOM

Funções

Antes de explicar como se dispara um evento DOM você deve saber que funções em JavaScript são conjuntos de códigos que serão executados só quando um evento ocorrer. Para fazer uma função você vai colocar o código da função dentro de bloco. Um bloco rm JS fica dentro de chaves "{}". A seguir vamos ver a estrutura de uma função:

function ação (parâmetro){Aqui coloca o código da função}

devemos colocar a palavra function para indicar que é uma função, logo após vamos nomer a função(a função tem que ser um verbo, ou seja, a função é uma ação), depois pode colocar um parâmetro dentro das aspas e por fim dentro das chaves (bloco) você irá colocar o código.
