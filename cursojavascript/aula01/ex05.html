Entendendo o DOM

DOM é Document Object Model. É um conjunto de objetos dentro do navegador que da acesso aos componentes do site

Árvore DOM

Tudo começa na raiz que é a janela do seu navegador ou seja a Window. Dentro do Window temos outros objetos como location, document(onde você está atualmente) e history(é o histórico)

Dentro do Document exite o HTML

Dentro de HTML temos mais dois objetos (ou filhos) que são o Head e body. HTML é um parent (pode chamar de pai ou mãe) de head e body, quem está embaixo é child e quem está em cima é parent é uma posição hierárquica.

Imagine que dentro de body eu tenho um paragrafo e dentro deste paragrafo está uma palavra envelopada com a tag strong, a tag strong é child do paragráfo e este é parent da tag

Obs: os scripts permitem que manipulemos elementos na árvore DOM

Navegando na árvore DOM

Há várias formas de acessar os elementos

  • Por marca
  • Por ID
  • Por nome
  • Por Classe
  • Por seletor

Forma correta de selecionar o DOM pelo script

O mais recomendado é pelo método Selector como:

querySelector() ou querySelectorAll()
