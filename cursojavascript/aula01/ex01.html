Variáveis

uma variável pode ser atribuida o valor null. EX: variavel1 = null

Usamar "var par indicar que é uma variavel e depois colocamos o nome. EX: "var n1=5"

OBS: Além de var podemos usar "let" para indicar que é uma variável

Quando formos armazenar uma string podemos colocar em três tipos de aspas: dupla =""; apostrofo = '' e crase: ``

Identificadores

Os nomes das variáveis saão chamados de identificadores e podem começar com letra,$ ou _

Não podem começar com números e é possível usar letras ou números

É possível usar acentos e símbolos

Não podem conter espaços

Não podem conter palavras reservadas(palavras utilizadas como comando pelo JS)
