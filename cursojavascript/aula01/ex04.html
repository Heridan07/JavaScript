Operadores JS

Operadores Aritméticos

+, -, *, / (divisão de números reais, ou seja, terá casa decimal), % (resto da divisão inteira), ** (potenciação)

Ordem de precedência: () -> ** -> *,/,% -> +, -

Operadores de incremento: n++ ou n--

Operadores Relacionais

  • >=
  • <=
  • >=
  • <=
  • == (esse é o simbolo de igualdade)
  • != (sinal de diferente)
  • Operador relacional de identidade restrita
obs: 5 == '5' é verdadeiro pois este operador não diferencia tipo, somente a grandeza do valor. Mas 5 === '5' seria falso pois o operador '===' diferencia tipo.

Operadores lógicos

  • ! (negação)
  • && (conjunção ou 'e lógico')
  • || (disjunção ou 'ou lógico')

Ordem de execussão: ! -> && -> ||

Operador Ternário

Formado por dois símbolos que aparecem na mems aexpressão: '?' e ':'.

É ternário por possuir três partes dentro desta estrutura: ' teste ? true : false '. O primeiro é uma acertiva, a segunda parte é o que ocorre se a acertiva for verdadeira e a terceira parte é o que ocorre se a acertiva for falsa.

Ex: média >=7.0 ? "Aprovado" : "Reprovado".
